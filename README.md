# rest-api-gin

## Description

A REST API written in [Go](https://go.dev/) for the [Alura Golang course](https://www.alura.com.br/formacao-go).

## Run

Create database containers with [Docker Compose](https://docs.docker.com/compose/).

```sh
docker compose up -d
```

Run the app.

```sh
go run main.go
```

## Test

```sh
go test
```
